# Guide d'utilisation — Studio Danse&CO

## Accès au Studio

**URL :** https://VOTRE-URL.vercel.app/studio
**Connexion :** votre compte sanity.io (email + mot de passe)

---

## Voir l'aperçu avant publication

1. Dans le Studio, cliquer sur l'icône **"Presentation"** dans la barre latérale gauche
2. Le site s'affiche en direct à droite de l'écran
3. Modifier un champ — le changement apparaît immédiatement dans l'aperçu
4. Cliquer **"Publish"** pour rendre le changement visible sur le site public

---

## Modifier le contenu

### Disciplines (cours proposés)

1. Cliquer **"💃 Disciplines"** dans le menu gauche
2. Sélectionner la discipline à modifier
3. Modifier le titre, la description ou les bénéfices
4. Cliquer **"Publish"**

### Planning des cours

1. Cliquer **"📅 Planning des cours"**
2. Sélectionner un cours existant, ou créer un nouveau avec le bouton **"New"**
3. Remplir : nom du cours, jour, heure, salle
4. Cliquer **"Publish"**

### Tarifs

1. Cliquer **"💰 Tarifs"**
2. Modifier les lignes de tarif (prix, libellé, détail)
3. Mettre à jour la saison si nécessaire
4. Cliquer **"Publish"**

### Professeurs

1. Cliquer **"👩‍🏫 Professeurs"**
2. Sélectionner un professeur ou créer avec **"New"**
3. Remplir : nom, spécialité, biographie, expérience, photo
4. Cliquer **"Publish"**

### Actualités

1. Cliquer **"📢 Actualités"**
2. Cliquer **"New"** en haut à droite
3. Remplir : titre, date, résumé, image, lien (optionnel)
4. Cocher **"Publié"** pour rendre l'actualité visible
5. Cliquer **"Publish"**

### Salles

1. Cliquer **"🏛️ Salles"**
2. Sélectionner une salle ou créer avec **"New"**
3. Remplir : nom, adresse, description, URL Google Maps
4. Cliquer **"Publish"**

### Informations générales (téléphone, email, réseaux sociaux)

1. Cliquer **"⚙️ Informations générales"**
2. Modifier les champs souhaités
3. Cliquer **"Publish"**

---

## Délai de mise à jour

Les modifications sont visibles sur le site dans les **10 secondes** après publication.
