# Danse & CO - Tech Stack

## Core Technologies
-   **Framework:** [React](https://react.dev/) (v18+) with [TypeScript](https://www.typescriptlang.org/) for a robust, type-safe development experience.
-   **Build Tool:** [Vite](https://vitejs.dev/) for extremely fast development and optimized production builds.
-   **Styling:** **Vanilla CSS** (using modern features like CSS Variables, Flexbox, and Grid) to maintain a lean, high-performance footprint without the bloat of a CSS framework.
-   **Icons:** [Lucide React](https://lucide.dev/) for clean, consistent, and lightweight iconography.

## Key Libraries
-   **Routing:** [React Router](https://reactrouter.com/) for single-page application (SPA) navigation.
-   **State Management:** Simple `useState`/`useContext` (if needed) for local state, keeping the application lightweight.
-   **Forms:** Standard HTML5 forms with basic React-controlled state.
-   **Animation:** [Framer Motion](https://www.framer.com/motion/) for subtle, elegant transitions and scroll animations that add a "premium" feel.

## Architecture
-   **Component-Based:** Logic and styles encapsulated within functional React components.
-   **Responsive First:** Mobile-first CSS design strategy.
-   **Modular CSS:** One `.css` file per component (e.g., `Hero.module.css` if using CSS Modules) for easier maintenance.
