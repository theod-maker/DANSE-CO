<objective>
## Goal
Ajouter des breakpoints `sm:` sur tous les titres h1/h2 pour éviter les débordements sur mobile (< 480px).

## Purpose
Sur iPhone SE (375px), les titres `text-5xl` (3rem) sont trop larges et forcent un scroll horizontal ou compressent le layout. Ajouter `text-3xl sm:text-5xl` comme base corrige tous les cas d'un coup.

## Output
10 fichiers modifiés — tous les titres h1/h2 ont un rendu correct dès 375px.
</objective>
