<objective>
## Goal
Créer tous les types de blocs Sanity + le schema `page` qui les contient. C'est la fondation du page builder.

## Purpose
Chaque page du site devient un document Sanity avec un champ `blocks` (array de blocs). Le client peut ajouter, supprimer et réordonner les blocs.

## Output
- 14 schemas de blocs dans `sanity/schemaTypes/blocks/`
- 1 schema `page` avec champ `blocks`
- Studio Sanity mis à jour avec section "Pages"
</objective>
