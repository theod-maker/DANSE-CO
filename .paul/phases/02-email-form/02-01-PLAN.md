<objective>
## Goal
Brancher le formulaire de contact sur Formspree pour que les messages soient réellement envoyés par email au professeur.

## Purpose
Le formulaire actuel simule l'envoi avec un `setTimeout`. Le professeur ne reçoit aucun message. C'est le seul bloquant fonctionnel avant livraison client.

## Output
1 fichier modifié. Le formulaire envoie un vrai email à chaque soumission. Confirmation visuelle en cas de succès ou d'erreur réseau.
</objective>

    ```
    const handleSubmit = async (e: FormEvent<HTMLFormElement>) => {
      e.preventDefault()
      setStatus('sending')
      try {
        const response = await fetch('FORMSPREE_ENDPOINT', {
          method: 'POST',
          body: new FormData(e.currentTarget),
          headers: { Accept: 'application/json' },
        })
        setStatus(response.ok ? 'success' : 'error')
      } catch {
        setStatus('error')
      }
    }
    ```
    Remplacer FORMSPREE_ENDPOINT par l'URL fournie au checkpoint.
